### Step 1: Domain Registration
- Domain: **elevatree.in**
- Status: ‚úÖ Ready for configuration

### Step 2: DNS Configuration (Update with Your Hosting Provider)

Point your nameservers to your hosting provider and add the following DNS records:

#### Required A Records:
```
Type: A
Name: @
Value: [Your Server IP]
TTL: 3600

Type: A
Name: www
Value: [Your Server IP]
TTL: 3600
```

#### Optional CNAME Records (for www subdomain):
```
Type: CNAME
Name: www
Value: elevatree.in
TTL: 3600
```

#### MX Records (for email):
```
Type: MX
Name: @
Value: mail.elevatree.in (if you have email hosting)
Priority: 10
TTL: 3600
```

### Step 3: SSL/HTTPS Configuration

Obtain an SSL certificate:
- **Recommended**: Let's Encrypt (free)
- **Auto-renewal**: Set up auto-renewal
- **Wildcard**: Get wildcard certificate for subdomains

Add DNS record for ACME challenge:
```
Type: TXT
Name: _acme-challenge.elevatree.in
Value: [Generated by your SSL provider]
```

### Step 4: Subdomain Setup (Optional)

If you want subdomains:
```
Type: CNAME
Name: blog
Value: elevatree.in

Type: CNAME
Name: api
Value: elevatree.in
```

### 3. Server Configuration

#### For Apache Hosting:
1. Ensure `.htaccess` is in document root
2. Enable mod_rewrite: `a2enmod rewrite`
3. Enable mod_deflate: `a2enmod deflate`
4. Enable mod_expires: `a2enmod expires`
5. Enable mod_headers: `a2enmod headers`

```bash
# On your server
a2enmod rewrite
a2enmod deflate
a2enmod expires
a2enmod headers
systemctl restart apache2
```

#### For Nginx Hosting:
Create nginx config:
```nginx
server {
    listen 80;
    server_name elevatree.in www.elevatree.in;
    
    # Redirect to HTTPS
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl http2;
    server_name elevatree.in www.elevatree.in;
    
    # SSL configuration
    ssl_certificate /path/to/certificate.crt;
    ssl_certificate_key /path/to/key.key;
    
    # Root directory
    root /var/www/elevatree.in/public;
    
    # Gzip compression
    gzip on;
    gzip_types text/plain text/css text/js text/xml text/javascript 
               application/javascript application/x-javascript application/xml+rss;
    
    # Security headers
    add_header X-Content-Type-Options "nosniff" always;
    add_header X-Frame-Options "SAMEORIGIN" always;
    add_header X-XSS-Protection "1; mode=block" always;
    
    # SPA routing
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    # Caching
    location ~* \.(jpg|jpeg|png|gif|ico|css|js)$ {
        expires 365d;
        add_header Cache-Control "public, immutable";
    }
}
```

## üÜò Troubleshooting Deployment

### Domain not resolving?
1. Check DNS records are set correctly
2. Wait 24-48 hours for DNS propagation
3. Use `nslookup elevatree.in` to verify
4. Check with your registrar

### HTTPS certificate not working?
1. Verify certificate is valid
2. Check certificate paths in nginx/Apache config
3. Restart web server
4. Use SSL Labs to test: https://www.ssllabs.com/ssltest/

### Pages showing 404?
1. Check .htaccess/nginx rewrite rules
2. Verify index.html is in document root
3. Check file permissions (755 for folders, 644 for files)
4. Clear browser cache

### robots.txt/sitemap not found?
1. Verify files exist in document root
2. Check file permissions
3. Verify web server is serving static files
4. Check .htaccess doesn't block these files
